# Product Sense

> 최종 업데이트: 2026-04-02

## Vision

초보운전자가 주차장 난이도를 한눈에 파악하고, 자신에게 맞는 주차장을 선택할 수 있는 서비스.

기획서: https://www.notion.so/3152d9c5abbf81e39f71c5ab21cd35ec

## Core Value

> "이 주차장, 초보도 괜찮을까?" 에 대한 답을 제공한다.

- 경쟁 서비스: 요금/위치 중심 (주차장 찾기)
- 쉬운주차: **난이도** 중심 (주차장 고르기)

## Difficulty Rating System

높은 점수 = 초보자에게 쉬운 주차장.
5개 항목: 진입로, 주차면 크기, 통로 여유, 출차 난이도, 종합 추천도. 각 1-5점.

| Score | Icon | Label | 설명 |
|-------|------|-------|------|
| 4.0-5.0 | :blush: | 초보 추천 | 넓고 여유로움 |
| 2.5-3.9 | :slight_smile: | 보통 | 일반적, 약간의 주의 |
| 1.5-2.4 | :skull: | 주의 | 좁거나 복잡함 |
| 1.0-1.4 | :skull::skull: | 초보 비추 | 매우 좁거나 기계식 |

## Scoring Philosophy

1. **데이터 소스 다양성**: 유저리뷰 + 웹 텍스트(블로그/카페/YouTube) 통합
2. **Bayesian 접근**: 데이터 부족 시 structural prior(3.0)로 수렴 — 과신 방지
3. **감성 x 관련도**: 확실한 추천/비추만 점수에 강하게 반영, 애매한 언급은 약한 영향

## Key Features

| Feature | Status | 설명 |
|---------|--------|------|
| 난이도 지도 | Live | 지도에서 난이도 아이콘으로 주차장 표시 |
| 주차장 상세 | Live | 리뷰, 블로그 포스팅, 영상 탭 |
| 사용자 리뷰 | Live | 5개 항목 별점 + 텍스트 |
| 위키 (인기 주차장) | Live | 웹 언급 많은 주차장 목록 |
| 자동 크롤링 | Live | 매시간 웹소스 수집 + AI 필터 + 매칭 |
| 지도 클러스터링 | Live | SuperCluster 기반 마커 그룹핑 |
| 콘텐츠 신고 | Live | 부적절 콘텐츠 신고 + 어드민 처리 |

## User Personas

1. **초보운전자** (핵심): 면허 취득 1년 이내, 주차 불안감 높음
2. **일반운전자**: 처음 가는 곳의 주차 난이도 사전 확인
3. **주차장 리뷰어**: 경험 공유를 통한 커뮤니티 기여
