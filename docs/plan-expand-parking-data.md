## Phase 1: 네이버 지역 검색 API (완료)

### API 스펙
- **URL**: `https://openapi.naver.com/v1/search/local.json`
- **인증**: `X-Naver-Client-Id` / `X-Naver-Client-Secret`
- **제한**: display 최대 5건, 일 25,000회 호출
- **응답**: title, address, roadAddress, mapx, mapy, telephone, category

### 검색 전략
시군구 229개 x 접미사 2개 ("주차장", "공영주차장") = 458 API 호출

### 실행 결과
| 항목 | 값 |
|------|-----|
| API 호출 | 458회 |
| 신규 추가 | 1,306건 |
| 중복 스킵 | 632건 |
| 소요 시간 | ~2분 |

### 중복 제거
- 기존 DB 전체를 메모리 로드 후 (이름 exact match + 좌표 200m 이내) 체크
- 새 항목은 `NV-{mapx}_{mapy}` 형태 ID로 INSERT

### 구현 파일
- `scripts/import-naver-local.ts` — 수집 스크립트
- 재실행 가능 (진행상황 `scripts/naver-local-progress.json` 저장)
